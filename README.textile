h1. Buddy List

h1. Historia
	
	O buddy list é o primeiro projeto do grupo de estudos do Hora extra SP

h1. Instalação

	Ruby 1.8.7
	Rails 3 RC
	
	#Configurando o projeto em 5 passos
	
	1 - Faça um clone do projeto no github 
		git clone git@github.com:horaextrasp/buddy_list.git

	2 - Instale o bundler para instalar as gems do projeto
		gem install bundler --pre (--pre server para instalar a versão Release Candidate)
		
	3 - Rode o bundler para instalar as gems do gemfile
		bundle install
		
	4 - Crie o banco de dados.
		rake db:create:all
		
	5 - Crie as tabelas.
		rake db:migrate
		
	6 - Prepare seu banco de testes
		rake db:test:prepare
